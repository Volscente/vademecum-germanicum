## Problem

The UI of the app is very basic at the moment, both in terms of color palette and appealing.

## Approach direction

- Keep the current color palette based on "green", but make the dark mode less "bright green"
- The appearance of the UI should be more clean and lean
- Adopt a Linux-style or MacOS style for the UIs
- Add a logo

## Success criteria

- Color palette remains the same
- UI is cleaner and leaner
- All the usability functions (e.g., Add word, enrich word, etc.) remain the same

## Constraints

- Color palette should remain the same. Only slightly changes to it are allowed. However the "green" should remain the central focuss
